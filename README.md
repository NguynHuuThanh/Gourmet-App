# Gourmet-App
- This web application is for reviewers who have experienced at restaurents will share for other users to know about the restaurent.

## Tech Stack
### Frontend: 
  HTML, CSS, Javascript
### Backend: 
  PHP
### Database: 
  MySQL
### Design Pattern: 
  Model–View–Controller (MVC)

## Feature
- Users group: Log in, Add new post, Edit post, and Delete post
- Guest group: View all posts and brief of each post, Register account.
